## Set-Up
1. install dependencies with:
```
npm i
````

2. Place Documents to send in sourceUtils folder and adjust file name and public/private keys

3. Run patient side with:
```
ts-node patient.ts
````
4. Run research side with:
```
ts-node research.ts
````
